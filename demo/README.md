# demo

本项目为 Spring Boot + MyBatis-Plus 示例工程。

## 主要技术栈

- Spring Boot 2.7.18
- MyBatis-Plus 3.5.5
- Lombok 1.18.32
- MySQL 8.0.33

## 快速开始

1. 配置数据库连接（见 `src/main/resources/application.yaml`）。
2. 构建并运行：
   ```sh
   ./mvnw spring-boot:run
   # 或
   mvn spring-boot:run
   ```

## 目录结构

- `src/main/java` 业务代码
- `src/main/resources` 配置文件
- `src/test/java` 测试代码

## 其他

如需更多帮助，请参考 Spring Boot 官方文档。
